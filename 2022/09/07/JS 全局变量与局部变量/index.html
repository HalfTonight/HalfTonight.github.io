全局变量与window上的变量区别：

  1. 所有 JavaScript 全局对象、函数以及变量均自动成为 window 对象的成员。
  2. 全局变量是 window 对象的属性。
  3. 全局函数是 window 对象的方法
  4. windown上定义的变量可以通过delete删除，而直接定义的全局变量是不能删除的
